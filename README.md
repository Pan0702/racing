DirectXとC++を用いて個人で開発したゲームです。ゲーム内容としては、UFOが人間に見られないように動物を吸い込み、吸い込んだだけUFOの力が強くなっていくゲームです。
学校で用意されたライブラリ（フレームワーク）を使用しています。

特に工夫し、こだわった部分です。

透視除算による正確な3D変換、EPSILON値を使った数値安定性の確保、Y=0平面への最適化により、堅牢で高速な座標変換を実現しています。 

操作説明です。
開始直後Mを押したらエディターに切り替わります。
操作説明
マウスのころころ＋Shift：上下移動
マウスのころころ＋Ctrl：左右移動
マウスのころころ＋Alt：拡大縮小
マウスのころころ：オブジェクトの回転
左クリック：つかむ、はなす
